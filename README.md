# Hana & Butrint Wedding Invite

Static one-page wedding invitation for GitHub Pages.

## Preview

Open `index.html` directly in a browser, or run a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy on GitHub Pages

1. Push this folder to `https://github.com/latifajrona/dasma-hana-butrint.git`.
2. In GitHub, open repository settings.
3. Go to Pages.
4. Set source to the `main` branch and root folder.

The site uses only HTML, CSS, JavaScript, and local images.
